# Modern Login UI

A sleek, responsive, and visually engaging user authentication interface built from scratch. This project showcases modern CSS layout techniques, interactive UI components, and a clean, user-centric design language.

## 🚀 Features

* **Dynamic Floating Labels:** Utilizes modern CSS pseudo-classes to create smooth, animated input transitions.
* **Interactive Form Elements:** Includes a custom styled password visibility toggle icon and "Remember Me" checkbox.
* **Social Authentication Layout:** Pre-configured UI buttons for Google, Apple, and Facebook sign-in options.
* **Fluid Background Aesthetics:** Layered SVG vector blobs that give the form a contemporary, artistic depth.
* **Fully Responsive:** Optimizes seamlessly across mobile, tablet, and desktop viewports

## 🛠️ Tech Stack

* **HTML5:** Structured using clean, semantic elements.
* **CSS3:** Built with Flexbox/Grid layouts, custom variables, and smooth transition animations.
